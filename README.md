# 🚀 TalentAI - AI Powered Resume Screening System

TalentAI is a Full Stack AI-Powered Recruitment Platform that automates resume screening, candidate evaluation, ranking, skill gap analysis, and hiring decision support.

The platform helps recruiters reduce manual screening efforts by generating intelligent candidate insights and match scores automatically.

---

## 📌 Features

### 📄 Resume Upload Center
- Upload resumes (PDF, DOCX, TXT)
- Automatic resume parsing
- Candidate profile generation
- AI confidence scoring

### 🤖 AI Candidate Intelligence
- AI-generated candidate summaries
- Match score calculation
- Strengths & weaknesses analysis
- Career growth prediction
- Leadership potential scoring
- Communication assessment
- Culture fit analysis

### 👥 Candidate Management
- Candidate database
- Candidate profiles
- Search and filtering
- Status tracking

### 🏆 Candidate Ranking
- Intelligent candidate ranking
- Match score comparison
- Talent prioritization

### ⚖️ Comparison Matrix
- Side-by-side candidate comparison
- Skill comparison
- Hiring decision support

### 📊 Skill Gap Analysis
- Missing skills detection
- Weak skill identification
- Learning roadmap recommendations
- Certification suggestions

### 🎤 Interview Copilot
- AI-generated interview questions
- Technical assessment support
- Candidate-specific interview preparation

### 🛡️ Fraud Detection
- Risk indicators
- Profile validation
- Fraud risk scoring

### 📈 Analytics Dashboard
- Recruitment insights
- Candidate statistics
- Hiring analytics
- Performance metrics

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/TalentAI.git
```

### Navigate to Project

```bash
cd TalentAI
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
